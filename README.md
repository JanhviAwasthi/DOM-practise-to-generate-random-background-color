# 🎨 Dynamic Color Changer

A simple and interactive web project that allows users to dynamically change the background color and modify the title size using buttons.

---

## 🚀 Live Demo

🔗 [Click here to view the project](http://127.0.0.1:5500/index.html)

> ⚠️ Note: This link works only on your local machine. To make it accessible online, deploy using GitHub Pages or Netlify.

---

## 📌 Features

* 🎯 Change background color with a single click
* 🔠 Increase title font size dynamically
* 🔄 Reset everything back to default
* 💡 Beginner-friendly JavaScript DOM manipulation project

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

📁 Dynamic-Color-Changer

---

## ⚙️ How It Works

* **Change Background Color Button** → Changes the page background to blue
* **Increase Title Size Button** → Enlarges the heading text
* **Reset Button** → Restores default styles
